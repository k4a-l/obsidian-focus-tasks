# Focus Tasks 仕様書

## 1. アクティブノートタスク抽出 (active-note)

- アクティブノートからタスクを抽出し、サイドペインへ一覧表示。
- フィルター（詳細アコーディオン）でステータス別の表示切り替え。
- 一覧のタスククリックで、エディタ上の該当行へジャンプ。

***

## 2. デイリーノートトラッキング (daily-note)

今日のデイリーノートのタスクを解析し、アクティブエディタ上部にタイマー付きバナーを表示。

### 2.1 デイリーノート設定

- `.obsidian/daily-notes.json` の設定値（ folder, format ）を自動同期。

### 2.2 抽出およびソート

- デイリーノートに記述されたタスクのうち、以下の条件に合致するものを抽出。
- 抽出されたタスクは、デイリーノート内の記述順に関わらず、開始時間（ HH:mm ）の昇順（早い時間順）で自動ソートされてバナーに表示される。

### 2.3 タスクの判定条件と表示状態

- 完了済みのタスク（ `[x]` または `[X]` ）は除外
- 進行中状態:
  - 条件: 開始時刻を過ぎており、かつ範囲内または警告閾値（設定可能）未満の未完了タスク。
  - 表示: アイコンは「 ⏱️ 」、テキストとタイマーは通常色。
  - タイマー: 開始時間からの経過時間（例: 23m または 1h 23m 形式）。
- 警告状態 (is-warning):
  - 条件: 終了時間を過ぎているか、または範囲なしタスクで開始時間から設定された警告閾値（分）を経過している未完了タスク。
  - 表示: アイコンは「 ⏳ 」、テキストと時間範囲は警告色（オレンジ）、タイマーはエラー色（赤）。
  - タイマー: 開始時間からの経過時間。
- 予告状態 (is-upcoming):
  - 条件: 開始時間前だが、設定された予告リードタイム（分）以内にある未完了タスク。
  - 表示: アイコンは「 🔔 」、全体が薄いグレー。
  - タイマー: 開始時間までの残り時間（例: -25m 形式）。

### 2.4 バナー・タイマー動作仕様

- 0件時:「📅 取り組んでいるタスクはありません」を表示。このバナー自体をクリックすると、今日のデイリーノートが開く（既存のタブがあればそこへ移動）。
- 複数件時: 時系列順に縦に並べて複数行表示。各タスク行をクリックすると、すでにデイリーノートを開いているタブがあればそこに遷移し、なければ開いた上で、該当タスクの行へ自動スクロールしてフォーカス。
- タイマー更新: 10秒ごとに更新。ちらつき防止のため、タスク構成の変化時のみDOMをクリアして再生成し、時間更新はテキストノードの差分書き換えのみで実行。

### 2.5 設定項目 (Settings)

- Warning threshold for tasks without range (minutes) (デフォルト: 60分)
  - 範囲指定のないタスクが自動的に警告（閉じ忘れ）状態に切り替わるまでの時間。
- Upcoming task lead time (minutes) (デフォルト: 30分)
  - 開始前のタスクを予告（ 🔔 ）表示し始めるまでの先行時間。

### 2.6 時間操作コマンド (Toggle/Update task time)

カーソル行に対して実行。タスク形式でない行は、タスク化して時間を適用。

| 元の書式 | 実行後の書式 | 挙動説明 |
| :--- | :--- | :--- |
| `テキスト` | `- [ ] HH:mm テキスト` | タスク化＋開始時間挿入 |
| `- リスト` | `- [ ] HH:mm リスト` | リストをタスク化＋開始時間挿入 |
| `- [ ] タスク` | `- [ ] HH:mm タスク` | 開始時間挿入 |
| `- [ ] HH:mm タスク` | `- [x] HH:mm - HH:mm タスク` | 終了時間追加で完了にする |
| `- [ ] HH:mm - HH:mm タスク` | `- [x] HH:mm - HH:mm(新) タスク` | 終了時間を現在時刻で更新して完了にする |
